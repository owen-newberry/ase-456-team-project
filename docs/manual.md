## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/owen-newberry/ase-456-team-project.git
   ```

2. Navigate to the app directory:
   ```bash
   cd ase-456-team-project/app
   ```

3. Install dependencies:
   ```bash
   flutter pub get
   ```

4. Run the app:
   ```bash
   flutter run
   ```

### System Requirements

- Flutter SDK 3.0+
- Dart SDK 2.17+
- iOS 12+ or Android 5.0+
- Internet connection for API access

## Account Management

### Creating an Account

1. Launch the app
2. Tap **Sign Up** on the login screen
3. Enter your email address
4. Create a password (minimum 6 characters)
5. Tap **Create Account**

### Logging In

1. Enter your registered email
2. Enter your password
3. Tap **Log In**
4. You'll be redirected to the home screen

### Signing Out

1. Tap the **Profile** icon in the top navigation bar
2. Scroll to the bottom of your profile
3. Tap **Sign Out**
4. Confirm your action

### Deleting Your Account

1. Go to your **Profile** page
2. Tap **Delete Account**
3. Confirm the deletion
4. Your account and all associated data will be permanently removed

## Browsing Content

### Home Screen

The home screen displays multiple sections:

- **Suggested Movies/TV Shows** ‚Äî Personalized recommendations
- **Best Recent Releases** ‚Äî Highly-rated new content
- **Movie of the Day** ‚Äî Daily featured selection
- **Genre Categories** ‚Äî Action, Comedy, Horror, Drama, Animation, Romance

### Switching Between Movies and TV Shows

1. Locate the **mode toggle** button in the app bar (movie/TV icon)
2. Tap to switch between Movies and TV Shows mode
3. All content sections will update to reflect your selection

### Scrolling Through Content

- Swipe **horizontally** within each category row to see more titles
- Swipe **vertically** to move between sections
- Tap any poster to view details

## Search

### Searching for Content

1. Tap the **search icon** (üîç) in the app bar
2. Type your search query in the text field
3. Results update as you type
4. Tap a result to view details

### Search Tips

- Search is case-insensitive
- Partial titles work (e.g., "dark" finds "The Dark Knight")
- Special characters are supported (e.g., "Spider-Man: No Way Home")
- Clear the search field to return to the home view

## Movie & TV Show Details

### Viewing Details

Tap any movie or TV show poster to open the detail view, which includes:

- **Rating** (out of 10)

- **Full-size poster**

- Tap the **back arrow** in the top-left corner

## Reviews & Tracking

### Marking Content as Watched

1. Open the movie/TV show details
2. Tap the **"Watched"** button
3. The item is added to your watched list

### Writing a Review

1. Open the movie/TV show details
2. Tap **Write Review**
3. Select a rating (1-10 stars)
4. Write your review text
5. Tap **Submit**

### Viewing Your Reviews

1. Go to your **Profile** page
2. Your reviews are listed under **My Reviews**
3. Tap any review to edit or delete it

## Personalization

### Dark Mode / Light Mode

1. Tap the **theme toggle** button (sun/moon icon) in the app bar
2. The app immediately switches between light and dark themes

### Theme Features

**Light Mode:**
- White/light gray backgrounds
- Dark text
- Suitable for daytime use

**Dark Mode:**
- Dark gray/black backgrounds
- Light text
- Reduces eye strain in low-light conditions

## Troubleshooting

### App Won't Load Content

1. Check your internet connection
2. Pull down to refresh the page
3. Close and reopen the app
4. Check if TMDB API is accessible

### Login Issues

1. Verify your email is correct
2. Check your password (case-sensitive)
3. Try the "Forgot Password" option
4. Create a new account if needed

### Images Not Loading

1. Check your internet connection
2. Wait a moment ‚Äî images load progressively
3. Some older titles may not have posters

### Search Not Working

1. Check your spelling
2. Try a shorter search term
3. Ensure you're searching in the correct mode (Movies vs TV)

### App is Slow

1. Close other apps to free memory
2. Check your internet speed
3. Try restarting the app
